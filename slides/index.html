PART 3: Feature detection

  • Safely add new features
  • Polyfills

Example: Geolocation

  • Make location-aware HTML5 apps
  • API: navigator.geolocation.getCurrentPosition
  • Callback: position.coords lat/lon

Mozernizr and Polyfills

  • Compatibility tables: caniuse.com
  • Modernizr does feature detection
  • Polyfill: browser functionality patches
  • Examples of polyfills: long list
  • Conditional polyfill loading: yepnope

Build it!

Make your TODO app location aware. Be sure to feature detect to ensure that the browser supports geolocation. For bonus points, use yepnope to load the polyfill if there's no geolocation support.

  • Feature detection: Modernizr
  • Conditional loading: Yepnope
